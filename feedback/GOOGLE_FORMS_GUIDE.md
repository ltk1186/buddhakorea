# 📝 Google Forms 피드백 설정 가이드 (5분 완료!)

## 1단계: Google Form 만들기 (2분)

1. **[Google Forms](https://forms.google.com)** 접속
2. **➕ 빈 양식** 클릭
3. 폼 제목: **"Buddha Korea 피드백"**
4. 설명: **"소중한 의견을 들려주세요"**

## 2단계: 질문 추가하기 (2분)

다음 필드들을 추가하세요:

### 질문 1: 이름
- 질문 유형: **단답형**
- 질문: **이름 (선택사항)**
- 필수 항목: ❌ 체크 해제

### 질문 2: 이메일
- 질문 유형: **단답형**
- 질문: **이메일 (선택사항)**
- 필수 항목: ❌ 체크 해제

### 질문 3: 전화번호
- 질문 유형: **단답형**
- 질문: **전화번호 (선택사항)**
- 필수 항목: ❌ 체크 해제

### 질문 4: 메시지
- 질문 유형: **장문형**
- 질문: **메시지**
- 필수 항목: ✅ 체크 (필수!)

## 3단계: 이메일 알림 설정 (선택사항)

1. 우측 상단 **⋮** 메뉴 클릭
2. **응답** 탭으로 이동
3. **⋮** 메뉴 → **새 응답에 대한 이메일 알림 받기** 체크
   - 이제 누군가 피드백 제출하면 이메일로 알림 받음!

## 4단계: 폼 꾸미기 (선택사항)

1. 상단 **🎨 팔레트** 아이콘 클릭
2. **헤더 이미지**: Buddha 관련 이미지 추가
3. **테마 색상**: 보라색 선택 (사이트와 맞추기)
4. **글꼴**: 기본 또는 Noto Sans

## 5단계: 링크 가져오기 (30초)

1. 우측 상단 **전송** 버튼 클릭
2. **🔗 링크** 아이콘 클릭
3. **URL 단축** 체크 ✅
4. **복사** 클릭

얻은 링크 예시: `https://forms.gle/AbCdEfGhIjKl`

## 6단계: 웹사이트에 적용 (30초)

`js/feedback.js` 파일을 열고:

```javascript
// 이 부분을 찾아서
const GOOGLE_FORM_URL = 'YOUR_GOOGLE_FORM_URL_HERE';

// 여러분의 링크로 변경
const GOOGLE_FORM_URL = 'https://forms.gle/AbCdEfGhIjKl';
```

## ✅ 완료!

이제 피드백 버튼을 클릭하면 Google Form이 새 탭에서 열립니다.

## 장점 ✨

- **무료**: 완전 무료, 제한 없음
- **간단**: 5분이면 설정 완료
- **안정적**: Google 서버 사용
- **자동 저장**: Google 스프레드시트에 자동 저장
- **이메일 알림**: 새 피드백 시 즉시 알림
- **통계**: 자동으로 응답 통계 제공
- **모바일 최적화**: 자동으로 모바일 화면 최적화

## 응답 확인 방법

1. Google Forms 열기
2. **응답** 탭 클릭
3. 개별 응답 보기 또는
4. **📊 스프레드시트 만들기** 클릭하여 Excel처럼 관리

## 추가 팁 💡

### 응답 스프레드시트 자동 생성
1. 응답 탭 → **스프레드시트** 아이콘
2. 새 스프레드시트 생성
3. 이제 모든 응답이 자동으로 스프레드시트에 저장됨

### 폼 임베드 (선택사항)
만약 모달 안에 폼을 넣고 싶다면:
1. 전송 → **< >** HTML 아이콘
2. 너비/높이 설정
3. HTML 코드 복사
4. 웹사이트 modal에 iframe으로 삽입

## 문제 해결

**Q: 링크가 작동하지 않아요**
- URL이 정확히 복사되었는지 확인
- 따옴표 안에 정확히 넣었는지 확인

**Q: 이메일 알림이 안 와요**
- 스팸 폴더 확인
- Google Forms 설정에서 알림 활성화 확인

**Q: 폼을 수정하고 싶어요**
- Google Forms에서 언제든 수정 가능
- URL은 그대로 유지됨
