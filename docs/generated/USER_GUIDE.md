## Ephemeral Sentences

"Ephemeral Sentences" (ì°°ë‚˜ì˜ ë¬¸ì¥) is an interactive experience demonstrating impermanence (ë¬´ìƒ/anicca).

### How It Works

1. Open the experience from the landing page
2. Type anything in the text area
3. After a brief pause, your text transforms:
   - Characters turn golden
   - They fade away one by one
   - From last character to first

4. The canvas clears, ready for new input
5. Press **Escape** or click outside to close

### Customization

The effect can be configured in the source code:

| Setting | Default | Description |
|---------|---------|-------------|
| `FADE_DELAY` | 3000ms | Wait before fading starts |
| `USE_PARTICLE_EFFECT` | false | Alternative particle scatter |
| `PARTICLE_DISTANCE` | 150px | Scatter distance (if enabled) |

## Troubleshooting

### Common Issues

#### Text Input Not Working (Sutra Copying)

**Symptom**: Characters aren't appearing or are rejected

**Solutions**:
1. Ensure Korean IME is active (í•œ/ì˜ toggle)
2. Wait for character composition to complete
3. Verify you're typing the correct character
4. Try refreshing the page

#### Meditation Bell Not Playing

**Symptom**: No sound at meditation start/end

**Solutions**:
1. Click somewhere on the page first (enables audio)
2. Check device volume settings
3. Ensure browser has audio permissions

#### AI Chatbot Not Loading

**Symptom**: NotebookLM page shows error

**Solutions**:
1. Ensure you're logged into Google
2. Try a different browser
3. Disable ad blockers temporarily

#### Animations Lagging

**Symptom**: Choppy or slow animations

**Solutions**:
1. Close other browser tabs
2. Disable browser extensions
3. Use Chrome or Firefox for best performance
4. Enable "Reduce motion" in OS settings

## Tips for Practice

### Sutra Copying

- Practice in a quiet environment

### Meditation

- Choose a consistent time each day
- Start with shorter durations
- Use the bell sound as an anchor
